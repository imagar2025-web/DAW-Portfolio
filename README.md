## 📈 Progresión

El contenido sigue el orden del plan de estudios a lo largo de dos cursos:

**1º DAW** → Fundamentos de programación, HTML/CSS, bases de datos, entornos de desarrollo.

**2º DAW** → Desarrollo cliente y servidor, diseño de interfaces, despliegue, videojuegos y proyecto final.

Cada repositorio de módulo contiene los ejercicios y proyectos organizados por temas, de menor a mayor complejidad.
